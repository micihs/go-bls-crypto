# go-bls-crypto
A high level API for signing and verifying message digests and using BLS threshold signature scheme.

## Prerequisites

* [Barreto-Naehrig curves](https://github.com/dfinity/bn).
* Go 1.8 or later.

## Install

```sh
go get github.com/micihs/go-bls-crypto/bls
```
